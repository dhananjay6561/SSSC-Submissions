# 📝 React To-Do List

A simple and responsive To-Do List application built using React, Vite, and Tailwind CSS. It allows users to efficiently manage daily tasks with persistent storage using the browser's Local Storage.

## ✨ Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Automatic Local Storage persistence
- Responsive and clean UI
- Fast performance with Vite

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS
- UUID
- Local Storage API
