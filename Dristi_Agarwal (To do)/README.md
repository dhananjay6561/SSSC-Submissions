# Todo List App
This is a simple Todo List application built using React. It allows users to add tasks, mark them as complete or undo them, and remove tasks from the list.

## Features
- Add new tasks
- Mark tasks as complete or undo
- Remove tasks
- Display total number of tasks

## Technologies Used
- React
- JavaScript
- CSS

## What I Learned
While building this project, I learned how to:
- Use the useState hook
- Handle user input
- Add and remove items from a list
-  Display lists using map()
=>-based on state changes
